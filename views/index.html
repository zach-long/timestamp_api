Timestamp API - Node.js Microservice

A Node.js server which will accept a date value and return the date value in a natural language format and as a Unix timestamp.

View test cases:

Pass 01/23/4567 in natural language format - https://fierce-beyond-84088.herokuapp.com/January%2023%204567

Pass 01/23/4567 as a string - https://fierce-beyond-84088.herokuapp.com/01141994

Pass a unix timestamp - https://fierce-beyond-84088.herokuapp.com/946684800

Pass an invalid date (a thirteenth month) - https://fierce-beyond-84088.herokuapp.com/13%2030%202013 - I am currently working on more strict date validation.
